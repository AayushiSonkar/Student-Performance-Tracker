# Student Performance Tracker

## Project Description
Student Performance Tracker is a Python-based application that helps manage student academic records. It allows users to add, edit, delete, and view student marks while automatically calculating the average marks. The data is stored permanently using a JSON file.

---

## Features
- Add Student Marks
- Show Student Details
- Calculate Average Marks
- Edit Existing Marks
- Delete Subject Marks
- Reset All Marks
- Save Data in JSON File
- Load Previous Data Automatically
- Exception Handling for Invalid Input

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

